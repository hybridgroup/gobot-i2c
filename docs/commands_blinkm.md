# Functions

## FirmwareVersion

Returs the version of the current Frimware.

#### API Command

**FirmwareVersionC**

## Color

Returs the color of the driver.

#### API Command

**ColorC**

## Rgb(r, g, b)

Sets the RGB color.

#### Params

- **r** - Red
- **g** - Green
- **b** - Blue

#### Returns

- **(r,g,b)** - the LED color

#### API Command

**RgbC**

## Fade(r, g, b)

Fades the RGB color.

#### Params

- **r** - Red
- **g** - Green
- **b** - Blue

#### Returns

- **(r,g,b)** - the LED color

#### API Command

**FadeC**
